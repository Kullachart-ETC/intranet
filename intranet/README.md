# Intranet บริษัท

ระบบ Intranet สำหรับองค์กร ประกอบด้วย:
- Board ประชาสัมพันธ์
- ปฏิทินการทำงานพนักงาน
- ระบบจองห้องประชุม 4 ห้อง

## วิธี Deploy บน Railway

1. Push โค้ดขึ้น GitHub
2. ไปที่ railway.com → New Project → Deploy from GitHub
3. เลือก repository นี้
4. Railway จะ deploy อัตโนมัติ

## วิธีรันในเครื่อง (Local)

```bash
npm install
node server.js
```

เปิดบราวเซอร์ที่ http://localhost:3000

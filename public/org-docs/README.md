# เอกสารโครงสร้างองค์กร (Org Chart PDFs)

|---|---|

| ทีมผู้บริหาร | `management.pdf` |
| โครงสร้างบริษัท - Head Office | `structure-head-office.pdf` |
| โครงสร้างบริษัท - Branch | `structure-branch.pdf` |
| โครงสร้างบริษัท - Factory1 | `structure-factory1.pdf` |
| โครงสร้างบริษัท - Factory2 | `structure-factory2.pdf` |
